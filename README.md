# 粉墨数据台 — 京剧数据集沉浸式可视化系统

基于 1473 部京剧剧本数据，构建的"数字戏台"沉浸式可视化系统。

## 技术栈

| 技术 | 用途 |
|------|------|
| **React 18 + Vite 5** | 前端框架 |
| **Tailwind CSS 3** | 样式与主题（宣纸暖色京剧舞台风格） |
| **Flask 3.0** | 后端 REST API |
| **ECharts 5** | 统计图表（饼图/柱状图/桑基图/旭日图/热力图） |
| **D3.js 7** | 力导向关系图（Force-Directed Graph） |
| **Zustand** | 前端状态管理 |
| **Framer Motion** | 页面入场动画 + 首页脸谱浮动 |
| **Python + PyMuPDF** | PDF 数据提取与清洗 |

## 页面

| 路由 | 页面 | 功能 | 技术亮点 |
|------|------|------|----------|
| `/` | **开戏** | 幕布拉开动画 + 90 张脸谱匀速上浮 + 开场按钮 | Framer Motion、CSS 动画 |
| `/overview` | **数据总览** | 统计卡片 + 朝代柱状图 + 题材饼图 + 角色词云 | ECharts 交互点击跳转 |
| `/roles` | **角色之相** | Sunburst 行当层级 + 浮动子类筛选 + D3 力导向同现网络 | 旭日图 + D3 力导向 |
| `/melody` | **声腔之流** | 声腔统计卡片 + 桑基流向图 | ECharts Sankey |
| `/plays` | **剧目之脉** | 多维度组合筛选（朝代×题材×声腔）+ 剧目卡片 | useSearchParams 深度链接 |
| `/heritage` | **传承之路** | 朝代趋势折线（滑块刷选联）+ 题材×朝代热力图 | dataZoom + heatmap |
| `/face-generator` | **脸谱生成** | 角色搜索 → SVG 脸谱 + 颜色语义映射 | 行当→色相编码 |

## 可视化技术

| 技术 | 说明 |
|------|------|
| **D3 力导向图** | 角色同现关系网络（节点=角色、边=共同出演），支持拖拽/缩放/悬停高亮 |
| **Sunburst 旭日图** | 行当层级展示（生旦净丑 → 子类） |
| **交互式时间序列** | 朝代趋势折线 + 滑块刷选，联动过滤下方图表 |
| **热力图** | 题材×朝代交叉分析，点击联动跳转剧目筛选 |
| **词云** | 角色出演频率，点击跳转脸谱生成器 |
| **桑基图** | 声腔→板式 流向，点击跳转剧目筛选 |

## 交互特性

- **跨页面深度链接**：所有筛选状态通过 URL search params 同步，支持直接访问和分享
- **多维度组合筛选**：朝代、题材、声腔可同时筛选，`/plays?dynasty=三国&genre=武戏`
- **图表点击联动**：词云→脸谱、热力图→剧目、桑基图→剧目、柱状图→剧目筛选
- **URL 驱动 tabs**：`/roles?tab=network` 直接打开力导向图

## 数据

1473 个京剧剧本，提取为结构化 JSON：

| 文件 | 数量 | 说明 |
|------|------|------|
| `plays.json` | 1473 剧目 | ID、标题、朝代、题材、角色、声腔、剧情摘要 |
| `roles.json` | 3576 角色 | 名称、行当类型、类别、脸谱色 |
| `melodies.json` | 10 种声腔 | 名称、板式列表 |
| `relations.json` | 7876 条 | 角色-剧目关系 |

### 脸谱图像来源

首页背景上浮的 90 张京剧脸谱 PNG 图片来自 [AkikoOu/hqz-beijingoperaface-small](https://huggingface.co/datasets/AkikoOu/hqz-beijingoperaface-small) 数据集。原始 Parquet 文件包含 90 张 1250×1250 RGB 脸谱图，处理后去除白底、缩放到 120×120 带透明通道，置于 `frontend/public/faces/` 目录。

## 氛围效果

- **墨迹晕染光晕**：朱红/鎏金/靛蓝/墨色四团大型模糊色块缓慢漂移
- **金尘微粒**：20 颗金色光点从底部缓缓上升
- **暗角晕影**：边缘渐暗聚焦视觉
- **幕布帘头**：页面顶部朱红渐变装饰条
- **聚光灯效**：顶部中心暖金色径向渐变

## 设计风格

- **配色**：宣纸底、朱红主色、鎏金点缀、靛蓝辅助
- **字体**：Noto Serif TC（标题）、Noto Sans TC（正文）
- **卡片**：无填充透明毛玻璃，仅保留边框与阴影
- **竖排文字**：页面标题旁侧竖排装饰字（writing-mode: vertical-rl）

## 开发

### 1. 启动后端

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Flask 运行在 http://localhost:5000

### 2. 启动前端

```bash
cd frontend
npm install
npm run dev
```

Vite 运行在 http://localhost:5173（自动代理 `/api` → 后端）
